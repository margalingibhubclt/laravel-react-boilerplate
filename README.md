## Laravel React Redux Boilerplate

### Laravel react redux boilerplate is pre configured with

- Laravel 8
- Laravel Sanctum for SPA Auth
- React 17
- Redux 
- React Router
- Axios
- Ant Design
- Redux Saga
- SASS
- EsLint
- Preconfigured store, actions, sagas

### Preconfigured Modules

- User Login
- User Register
- Auth Routes
